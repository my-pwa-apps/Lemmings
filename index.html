How to Play

Guide lemmings from the entrance to the exit by assigning them different abilities.

Controls:

  • Click on a lemming to assign the selected ability
  • 1-8 Keys to quickly select abilities
  • P to pause the game
  • +/- to adjust game speed
  • D to toggle debug mode
  • ESC to return to level selection

Abilities:

  • Climber: Climb vertical walls
  • Floater: Fall safely from any height
  • Bomber: Explode after a countdown
  • Blocker: Block other lemmings
  • Builder: Build a diagonal bridge
  • Basher: Dig horizontally
  • Miner: Dig diagonally down
  • Digger: Dig straight down
